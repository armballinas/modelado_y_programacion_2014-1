Este es un ejemplo acerca de cómo leer líneas desde un archivo y cómo modificar el programa para que se lea desde la entrada estándar o desde un archivo cualquiera.

La intención de este ejemplo es que los alumnos puedan crear sus casos de prueba para resolver los problemas que un juez automático verifica.

Los archivos .txt son archivos de casos de prueba para el problema de Empaquetar Polígonos(packing polygons)

http://uva.onlinejudge.org/index.php?option=com_onlinejudge&Itemid=8&category=12&page=show_problem&problem=946

PRECAUCIÓN: El programa fue hecho en un linux fedora 18. Si al compilar o ejecutar en otro sistema operativo, principalmente uno que no sea tipo UNIX surgen errores, se debe consultar la documentación de dicho sistema para conocer la causa de estos.
